# Council extracts — working rules

> **Empty by design (for now).** No Council compromise text exists yet on file 2026/0138 (COD) — only
> the transmission cover note ST 10104/26. When the first Presidency compromise lands, it gets its own
> identically-named extract set (`ST-<nnnn>-<yyyy>_<slice>.md`, one file per `tracker.yaml`
> `extract_slices`), produced with the `transcribe-council-extract` skill.

Before adding or editing any file in this directory, read
[`_TRANSCRIPTION_GUIDE.md`](./_TRANSCRIPTION_GUIDE.md) and follow it. It is the canonical
standard for transcribing Council compromise-text extracts (source of truth, consolidated
reading, faithfulness rules, structure/anchors, header block, cross-links).
