# Related instruments

CADA is a **standalone regulation**: a text search of COM(2026) 502 finds no amending or repealing
provisions (confirm once more when transcribing Title V — final provisions). Unlike its package sibling
Chips Act 2.0 (which repeals Regulation (EU) 2023/1781), CADA only **interfaces** with existing law.
This page replaces the per-amended-instrument pages a tracker normally carries; split it into one page
per instrument if a compromise text ever introduces actual amendments.

| Instrument | CELEX no. | Interface with CADA |
|---|---|---|
| **Data Act** | 2023/2854 | Cloud-switching/interoperability baseline CADA builds on |
| **NIS2 Directive** | 2022/2555 | Defines the sectors whose public bodies are restricted to assurance levels 2–4; private NIS2-sector entities may run comparable impact assessments (Art 31) |
| **DORA** | 2022/2554 | Financial-sector ICT-risk regime referenced alongside the framework |
| **GDPR** | 2016/679 | Data-protection baseline; the Commission argues sovereignty goes beyond data transfers (EU–US DPF insufficient for operational autonomy) |
| **AI Act** | 2024/1689 | AI regulatory baseline for the leadership initiatives / frontier AI priority projects |
| **Cybersecurity Act** | 2019/881 | European cybersecurity certification ("substantial"+) feeds the level 2–4 audit criteria; note the pending CSA revision (COM(2026) 11), which CADA reportedly cross-references |
| **Interoperable Europe Act** | 2024/903 | Public-sector interoperability mandates referenced in the memorandum |

Source: explanatory memorandum of [COM(2026) 502](../../sources/README.md) and early legal analyses —
verify each interface against the operative text when the baseline is transcribed.
